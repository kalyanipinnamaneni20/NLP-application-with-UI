# NLP Application in Jupyter Notebook

This project is a simple NLP application with:
- File upload (.txt, .pdf, .jpg/.png)
- File size check (max 250 MB)
- Language detection
- Sentiment analysis
- Keywords extraction
- Named Entity Recognition (NER)
- Quick summary
